Zach's Portfolio Website

This week I built my personal portfolio website using HTML, CSS, and JavaScript. I added structure with semantic HTML tags, styled the page with a custom stylesheet, and added an interactive button using JavaScript.

Features

- Semantic HTML structure (header, main, sections, footer)
- Custom CSS for colors, layout, and spacing
- JavaScript button that changes text when clicked
- Personal image, interests list, and favorite website link
